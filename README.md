Проект Process Information Catcher (pid_catcher) представляет собой инструмент на Python для мониторинга и управления процессами операционной системы с использованием библиотеки psutil. Проект структурирован для обработки различных аспектов работы с процессами через командную строку.

## Основные компоненты

- ### cli_router.py: 
    Модуль для обработки аргументов командной строки и маршрутизации действий на основе переданных флагов.

- ### process.py: 
    Класс Process, представляющий информацию о процессе и методы для работы с ней, используя библиотеку psutil.

- ### process_counter.py: 
    Класс ProcessCounter, который подсчитывает количество процессов в различных состояниях (запущен, ожидание, остановлен и т.д.).

- ### show_all_process_service.py и sum_of_process_service.py: 
    Модули, предоставляющие сервисы для вывода информации о процессах и суммарной статистики через команды -a и -s.

## Как использовать

### Для запуска проекта выполните следующую команду:

```bash
python main.py [-a] [-s]
```

- Флаг -a или --all: Выводит информацию о всех процессах.
- Флаг -s или --sum: Выводит суммарную статистику по состояниям процессов.

## Зависимости

Для корректной работы проекта требуется установить библиотеки psutil и colorama:

```bash
pip install psutil colorama
```
